Primero se obtiene la frecuencia angular. Luego, como la velocidad máxima ocurre cuando |sin| = 1, se cumple vmax = ωnA.

Convierte el período a frecuencia angular ωn = 2π / T = 2π / 2.00 = 3.1416 rad/s
2
Calcula la amplitud con la velocidad máxima A = vmax / ωn = 0.100 / 3.1416 = 0.03183 m

Encuentra el ángulo de fase con la posición inicial cos(φ0) = x0 / A = 0.020 / 0.03183 = 0.6283 φ0 = cos-1(0.6283) = 51.07°

Calcula la aceleración máxima amax = ωn2A = 0.31416 m/s²
